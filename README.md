# Namma-Kelsa

## 📌 Overview
Namma-Kelsa is an Android application designed to connect daily wage workers with customers digitally.  
The platform helps skilled workers like painters, electricians, gardeners, plumbers, and laborers become easily discoverable without depending on middlemen.

This project focuses on creating a dignity-first self-employment platform with a simple and modern Android user experience.

---

## 🚀 Features

- 🔐 OTP Authentication
- 👤 Worker Registration & Profiles
- 📋 Job Listings
- 🔎 Worker Discovery
- ☁ Firebase Integration
- 🧭 Navigation Components
- 📱 Modern Android UI
- 🏗 MVVM Architecture
- 💉 Hilt Dependency Injection
